- Paleta fixa: base mint #cfe0d8, acentos roxo-escuro/navy, brinquedos rosa e amarelo. Sem tons frios/escuros.
- Personagens recorrentes em toda cena: boneco estilo Playmobil de roupa preta + boneca de vestido rosa com laço roxo.
- Desktop only — sem encodes mobile (`clipMobile`/`connectorsMobile` não são usados).
- Proibido texto/letras/logos legíveis dentro dos assets gerados.
